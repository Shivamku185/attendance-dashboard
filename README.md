# Attendance & Report Dashboard

## Project Overview
This repository contains attendance and reporting files used for data analysis and dashboard creation.  
The project focuses on organizing raw data and converting it into meaningful insights.

## Files Included
- **Attandance.xlsx** – Raw attendance data
- **YEARLY REPORT.xlsx** – Yearly summarized report
- **dashbord.pbix** – Power BI dashboard file

## Tools & Technologies
- Microsoft Excel
- Power BI
- Data Cleaning & Analysis

## Objective
- Track attendance data efficiently  
- Generate yearly performance reports  
- Visualize data using Power BI dashboards  

## How to Use
1. Download the Excel files for raw data reference
2. Open `dashbord.pbix` in Power BI Desktop
3. Refresh data if required
4. Analyze insights from visual dashboards

## Use Case
This project can be used for:
- Academic analysis
- Office attendance tracking
- Portfolio demonstration for Data Analyst roles
